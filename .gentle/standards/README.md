# Project Standards (Scream Architecture)

Esta carpeta contiene las convenciones arquitectónicas específicas de este proyecto. 
El orquestador (`sdd-orchestrator`) leerá estos archivos dinámicamente basándose en el `sdd.yaml` y los inyectará como contexto a los sub-agentes encargados de escribir o verificar código.

**Regla de oro:** Mantener estos archivos concisos. Explicar el "Qué" y el "Por qué", y dar un ejemplo del "Cómo".
